# Business Rules

## Normal Orders

- Customer can order any quantity.
- Fixed product prices apply.

## Bulk Orders

A bulk order is any order with more than 50 pieces.

Rules:
- Customer can request a custom price.
- Minimum allowed custom price is ₹7 per piece.
- Requests below ₹7 are not allowed.
- Final price is confirmed by the bakery through WhatsApp.

## Order Confirmation

- Every order is sent to WhatsApp.
- The bakery confirms the order before it is considered final.

## Pickup

- Customer selects pickup date and time.
- Order is prepared before the selected pickup time.
